# Portfolio

> My personal portfolio and interactive resume, built with React and Vite.

**Live site:** https://mo.seeleco.co

## Overview

A single-page portfolio with an animated background, a resume timeline, project showcases, and a contact form. Built as a fast, responsive React application and deployed on Vercel.

## Features

- **Animated hero:** an interactive particle/cosmos background.
- **Resume timeline:** experience, skills, and education at a glance.
- **Project showcase:** cards linking out to live projects.
- **Contact form:** wired to a form-handling service.
- **Responsive:** works across desktop and mobile.

## Tech Stack

- **Framework:** React 19 with Vite
- **Styling:** Tailwind CSS v4 (via the Vite plugin)
- **Deployment:** Vercel

## Getting Started

```bash
npm install
npm run dev      # start the dev server
npm run build    # production build to dist/
npm run preview  # preview the production build
```
